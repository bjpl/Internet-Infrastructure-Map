**Solution** (video_gen/input_adapters/document.py):
```python
def _clean_list_text(text: str) -> str:
    """
    Clean markdown list markers from text.

    Removes:
        - Leading/trailing whitespace
        - Markdown list markers (*, -, 1., etc.)
        - Extra spaces
        - Nested list indentation

    Args:
        text: Raw text that may contain markdown

    Returns:
        Cleaned text suitable for narration
    """
    # Remove markdown list markers
    text = re.sub(r'^\s*[-*+]\s+', '', text)
    text = re.sub(r'^\s*\d+\.\s+', '', text)

    # Remove extra whitespace
    text = ' '.join(text.split())

    # Remove nested list indentation
    text = re.sub(r'^\s+', '', text)

    return text.strip()
```
